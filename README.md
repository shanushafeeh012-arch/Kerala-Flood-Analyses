# Kerala Flood Analysis 🌊

## About
Exploratory Data Analysis and Flood Prediction
on Kerala Floods Dataset using Python.

## Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn (Naive Bayes)

## Results
- No missing values in dataset
- Model Accuracy: 79%
- Flood months: JUL, AUG, SEP

## Files
- `kerala.csv` - Dataset
- `notebook.ipynb` - Code

---

## How to Run

### Option 1 - Google Colab (Recommended)

1. Go to [colab.research.google.com](https://colab.research.google.com)
2. Click **File → Upload Notebook**
3. Upload the `notebook.ipynb` file
4. Run the first cell — a file upload button will appear
5. Upload `kerala.csv` when prompted
6. Click **Runtime → Run All**

### Option 2 - Kaggle Notebook

1. Go to [kaggle.com](https://kaggle.com)
2. Click **"Create Notebook"**
3. Click **"Add Input"** → search Kerala Floods Dataset
4. Copy paste the code cells one by one
5. Click **"Run All"**
